# TradeZone E-Commerce Data Analysis
 
## Project Overview

End-to-end SQL data analysis of TradeZone, a Nigerian e-commerce platform experiencing strong revenue growth but declining customer retention and seller quality issues. The project covers data cleaning and preparation, eight structured business analysis queries, and a formal analyst memo addressed to senior leadership.

## Repository Structure
 
| File | Description |
|------|-------------|
| `tradezone_project_postgresql.sql` | Part A — Full data cleaning script (PostgreSQL) |
| `tradezone_project_t2.sql` | Part A — Data cleaning script (MySQL version) |
| `analysis_queries_postgresql.sql` | Part B — 8 business analysis queries (PostgreSQL) |
| `tradezone_analyst_memo.docx` | Part C — Analyst memo addressed to Head of Growth and Head of Seller Operations |
